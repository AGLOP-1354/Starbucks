# 표(table)

position 속성

```plaintext
  값 | 의미 | 기본값
  -- | :--: | --:
  static | 기준 없음 | O
  relative | 요소 자신 | X
  absolute | 위치 상 부모 요소 | X
  fixed | 뷰포트 | X
```

| 값       |       의미        | 기본값 |
| -------- | :---------------: | -----: |
| static   |     기준 없음     |      O |
| relative |     요소 자신     |      X |
| absolute | 위치 상 부모 요소 |      X |
| fixed    |      뷰포트       |      X |
